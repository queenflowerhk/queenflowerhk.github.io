This repository is a static GitHub Pages site consisting of a single `docs/index.html` file and a `.nojekyll` marker. There are no dependency manifest files (no `package.json`, `go.mod`, `Gemfile`, `requirements.txt`, etc.), no vendoring directory, no lockfiles, and no build tooling. All third-party assets — CSS, JavaScript, fonts, images — are embedded directly in the HTML or served as static resources from the same repository. As such, there is no dependency management strategy to document.
