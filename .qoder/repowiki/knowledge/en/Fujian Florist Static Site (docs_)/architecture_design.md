Pure static site under `docs/` deployed to GitHub Pages. The entry point is `docs/index.html`, which loads Tailwind CSS via CDN, Google Fonts, Font Awesome, and the local `css/styles.css`. All runtime logic lives in five ordered `<script>` modules loaded from `js/`: `translations.js` (i18n store), `products.js` (fetches `products.json` at boot and renders category grids into DOM ids like `ceremonial-grid`), `cart.js` (in-memory cart state), `components.js` (shared UI helpers such as toast / navbar scroll), and `main.js` (orchestrates init, language switching, cart rendering, and WhatsApp checkout link generation). Product catalog data is externalised in `docs/products.json` keyed by category (`ceremonial`, `funeral`, `wreath`, `opening`, `association`, `graduation`, `pets`). A tiny Node dev server at `_serve.js` serves `docs/` over HTTP on port 8080 with MIME mapping so `fetch('products.json')` works locally; it also rewrites `/` to `/index.html` for SPA-style navigation. There are no build tools — assets are served directly.
