This repository is a GitHub Pages-hosted static marketing site for Fujian Florist. It contains only a single generated `docs/index.html` (89 KB) and a `.nojekyll` marker; there are no source files, build scripts, or application code. The HTML contains no logging calls (`console.log`, structured loggers, etc.) — it is purely a presentation layer served directly by GitHub Pages. Consequently, there is no logging framework, no log-level strategy, no structured fields, and no sinks to document.
